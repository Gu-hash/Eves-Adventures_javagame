About Me

Hi, my name is Grace and this is my first static webpage featuring a java game project.
The game is called "Eve's Adventures".
I made this project during my first year of university using a library provided by the university.
The game contains 3 levels each with a different story and is fully complete.
Below is a video showing me playing the game. I hope you enjoy watching!

How to play

Controls

Up arrow: Jump
Left arrow: Move left
Right arrow: Move right
[A] : Shoot lightning to the left
[D] : Shoot lightning to the right

Level 1

This is the easiest level! You must collect at least 4 apples while avoiding the snakes.
If the snakes touch you, you lose a life.
